Offset Explorer (formerly Kafka Tool) is a GUI application for managing and using Apache Kafka ® clusters. It provides an intuitive UI that allows you to quickly view objects within a Kafka cluster, as well as the messages stored in its topics. It contains features geared toward both developers and administrators. Some of the key features include:

  • Quickly view all your Kafka clusters, including their brokers, topics and consumers
  • View contents of messages in your partitions and add new messages
  • View offsets of consumers
  • Show JSON, XML and Avro messages in a pretty-printed format
  • Add and drop topics plus other management features
  • Save individual messages from your partitions to local hard drive
  • Import messages from local files into Kafka Topics using the Import Tool
  • Export messages from Kafka Topics to local files or another topic using the Export Tool
  • Generate test data for Kafka Topics using the Data Generator
  • Find messages across Topics and Partitions using the Find Messages Tool
  • Use the command-line tool for Kafka, Schema Registry, Kafka Connect, and message operations
  • Write your own plugins that allow you to view custom data formats
  • Offset Explorer runs on Windows, Linux and Mac OS
  • Support for Bulgarian, Chinese, Finnish, French, German, Greek, Italian, Japanese, Korean, Polish, Portuguese, Romanian, Spanish, Swedish, Turkish and Ukrainian
